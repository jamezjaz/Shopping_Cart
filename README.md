## About The Project

  This is an e-commerce application. It's a shopping cart application built with React &amp; Redux. It's a single page application. Here, users can add and remove items from the cart. It also shows the total cost of items added to cart.

### Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

 * [Node.js](https://nodejs.org/) is required to install dependencies and run scripts via npm.
 * A modern browser

### Clone
* Clone this repo:
  - Clone with SSH:
  ```
    git@github.com:jamezjaz/Shopping_Cart.git
  ```
    
 - cd to Shopping_Cart

### Setup
To install dependencies:

Run ```$ npm install```

To start the local webserver and have the application run live on http://localhost:3000/:

Run ```$ npm start```
